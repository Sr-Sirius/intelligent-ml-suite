Logistic Regression - Basic Concepts

What is Logistic Regression?

Logistic Regression is a supervised learning algorithm used for classification problems. It predicts the probability that an input belongs to a specific class, typically using a binary outcome such as 0 or 1.

Regression vs Classification

In regression, the output is a continuous value (e.g., price or temperature). In classification, the output is a category or class label (e.g., spam or not spam). Logistic Regression is used for classification, even though its name includes "regression".

Binary Classification

Logistic Regression is commonly used for binary classification, where there are only two possible outcomes:
  • Class 0 (e.g., No Fraud)
  • Class 1 (e.g., Fraud)

Target Variable / Class Label

The target variable is the output the model is trying to predict. In classification problems, this is also called the class label, such as "fraud" or "not fraud".

Features / Predictors

Features (or predictors) are the input variables used by the model to make predictions. These can include numerical or categorical data such as age, income, or transaction amount.

Probability Estimation

Logistic Regression predicts a probability value between 0 and 1. This value represents how likely an input belongs to a specific class.

Decision Boundary

The decision boundary is the threshold that separates different classes. It determines how the model classifies input data based on predicted probabilities.

Classification Threshold

The classification threshold is typically set at 0.5:
  • If probability ≥ 0.5 → Class 1
  • If probability < 0.5 → Class 0

Real-World Example

Logistic Regression is widely used in applications such as fraud detection, medical diagnosis, and spam filtering, where decisions must be made between two categories.
