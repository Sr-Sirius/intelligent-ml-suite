Bernoulli Naive Bayes - Application

Problem Description

This application aims to classify the risk level of a driver based on observable driving behaviors. Instead of using continuous values, this model evaluates whether certain risky behaviors are present or absent, such as overspeeding or phone usage while driving.

Dataset Explanation

The dataset is composed of binary variables representing driver behaviors. Each feature indicates whether a risky action occurred (1) or not (0). The target variable represents whether the driver is classified as high risk or low risk.

Input Variables and Target Variable

  • Input variables: overspeeding, night driving, phone usage, aggressive braking
  • Target variable: accident risk (1 = high risk, 0 = low risk)

Data Preparation

The dataset is structured using binary encoding, where each feature represents the presence (1) or absence (0) of a behavior. The data is then split into training and testing sets to evaluate the model’s performance.

Model Training Process

A Bernoulli Naive Bayes model is trained using labeled data. The algorithm calculates the probability of each class given the observed features, assuming independence between them. It then selects the class with the highest probability.

Interpretation of Results

The Bernoulli Naive Bayes model demonstrates solid performance in classifying driver risk based on binary behavior indicators. The accuracy reflects the model’s ability to correctly classify both high-risk and low-risk drivers.

Precision indicates that when the model predicts a driver as high risk, it is usually correct. Recall shows that the model successfully identifies most of the actual high-risk drivers, which is critical in safety-related applications.

The confusion matrix reveals some classification errors, which is expected due to the simplicity of the model and the independence assumption. Despite this, the model provides reliable and interpretable predictions.

The ROC curve and AUC value confirm that the model maintains a good balance between sensitivity and specificity, making it a strong baseline classifier for binary behavioral data.
