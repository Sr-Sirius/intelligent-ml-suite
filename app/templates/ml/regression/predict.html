Linear Regression — Application

This application predicts a user's expenses based on financial inputs such as income, previous expenses, and number of transactions. The goal is to provide an estimate that helps users understand their financial behavior.

Dataset Explanation

The dataset contains 50 records of financial data including income, previous expenses, number of transactions, and actual expenses. This data is used to train the regression model and identify patterns between variables.

Variables Used

Inputs: Income, Previous Expenses, Transactions
Output: Expense

Data Preparation

The dataset was cleaned and structured to ensure consistency. All variables were formatted numerically, allowing the regression model to process and learn from the data effectively.
