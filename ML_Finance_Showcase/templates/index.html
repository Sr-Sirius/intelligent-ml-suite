Expense Category Classifier

This page demonstrates how a Machine Learning model could classify expenses into Essential or Non-Essential categories.

Machine Learning Model Explanation

Problem: Many users overspend on non-essential purchases.

Goal: Automatically classify expenses to help users understand their spending behavior.

Algorithm

This system uses Logistic Regression, a supervised machine learning algorithm used for binary classification problems.

Binary classification means the model predicts one of two possible outputs:

  • Essential (1)
  • Non-Essential (0)

Model Inputs

  • Amount
  • Category
  • Description (optional)
  • Frequency

Model Output

The model predicts whether the transaction should be considered Essential or Non-Essential.

Why this is useful

  • Applies machine learning classification
  • Helps users understand spending behavior
  • Can later support recommendation systems
